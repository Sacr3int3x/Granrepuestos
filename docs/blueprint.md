# **App Name**: Auto Parts Hub

## Core Features:

- Home Page Carousel: Display a rotating carousel with high-resolution images and welcome message.
- Brand Showcase with Hover Effects: Showcase supported auto parts brands with logo display and interactive hover effects.
- Featured Products Grid: Display a grid of featured products on the home page for quick access.
- Parts Catalog (List/Grid View): Enable users to browse a comprehensive list or grid view of available auto parts, showing clear photos and key details.
- Advanced Filtering and Search: Implement robust search and filtering capabilities by brand, category (engine, suspension, body), and price range.
- Product Detail Page with Image Gallery: Provide a detailed product page with a high-resolution image gallery, technical specifications, and related parts.
- Admin CMS with Firebase Authentication: Secure admin interface for managing parts and brands, protected by Firebase Authentication.
- Firestore Integration: Use Firestore for structured storage of brands, categories and parts.

## Style Guidelines:

- Primary color: Dark navy blue (#243A73), evoking professionalism and trust, aligned with the automotive industry.
- Background color: Light gray (#E0E5EC), a neutral backdrop ensuring content clarity in the dark color scheme.
- Accent color: Electric blue (#477BFF), providing a vibrant highlight for calls to action.
- Body and headline font: 'Inter', a sans-serif for a clean, modern, objective feel.
- Use a consistent set of line icons for categories and interactive elements.
- Employ a responsive grid layout that adapts seamlessly to different screen sizes.
- Incorporate subtle transitions and hover effects to enhance user engagement.
